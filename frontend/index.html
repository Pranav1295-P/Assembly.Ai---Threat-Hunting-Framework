How it works

  1. Static intake — magic-bytes ID, hashes, entropy, PE/ELF parsing, packer hints, suspicious-API spotting.
  2. IOC extraction — URLs, domains, IPs, registry keys, file paths, mutexes, crypto wallets.
  3. ML classification — gradient-boosted model over 13 PE features predicts malicious probability.
  4. LLM investigation — Claude / OpenAI receives the structured dossier and reconstructs the attack chain.
  5. MITRE mapping — heuristic + AI techniques merged into a single coverage table.
  6. PDF report — executive summary, attack flow, process tree, IOC tables, recommendations.
